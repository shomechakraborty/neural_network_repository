# neural_network_repository

INTRODUCTION 

- This repository offers neural network regression and classification models that accommodate three optimizers - Stochastic Gradient Descent (SGD), Batch Gradient Descent (BDG), Mini-Batch Gradient Descent (MBGD) - layer configuration, and 9 additional hyperparameters. The models are designed to be fine-tuned to a user’s specific predictive application and preferences.
- This repository builds from a previous custom Stochastic Gradient Descent neural network regressor model that I developed for public health capacity forecasting research, available here: https://github.com/shomechakraborty/general-purpose-regression-neural-network.git
