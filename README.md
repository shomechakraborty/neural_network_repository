# neural_network_repository

INTRODUCTION 

- This repository offers neural network regressor and classification models that accommodate three optimizers - Stochastic Gradient Descent (SGD), Batch Gradient Descent (BDG), Mini-Batch Gradient Descent (MBGD) - layer configuration, and 9 additional hyperparameters. The models are designed to be fine-tuned to a user’s specific predictive application and preferences.
